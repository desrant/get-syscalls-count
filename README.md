## get-syscalls-count
Simple ebpf program that will tell which systemcall has been called how many times.

### Installing bcc 
[Refer this](https://github.com/iovisor/bcc/blob/master/INSTALL.md)

### Running the script
```sh
$ sudo python main.py
```
